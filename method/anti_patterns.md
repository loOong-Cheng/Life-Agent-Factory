# Anti-Patterns

## Generic Life Coach

The agent promises broad motivation, accountability, and advice without naming a specific loop.

Correction: extract one repeatable situation, such as "after dance class reflection" or "Sunday cooking plan review."

## Memory Hoarder

The agent stores everything because it might be useful later.

Correction: define memory types, retention rules, and consent points.

## Dashboard Too Early

The design jumps to metrics, charts, and tracking before the user knows what reflection matters.

Correction: begin with a lightweight ritual and one useful artifact.

## Productivity Translation

The agent turns an expressive or emotional desire into generic optimization.

Correction: preserve the user's original wording and ask what better would feel like.

## Overbuilt Persona

The agent is described as a wise mentor, coach, analyst, friend, archivist, and planner all at once.

Correction: decompose roles and assign each role to a specific ritual or command.

## Advice Without Evidence

The agent gives recommendations without having observed the user's actual loop.

Correction: start by collecting examples, logs, or reflections before proposing changes.
