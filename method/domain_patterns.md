# Domain Patterns

## Practice Lab

Used for dance, music, sport, language, drawing, or craft.

Common loop: practice session -> reflection -> pattern detection -> next focus.

Useful memory: drills, felt sense, teacher feedback, recurring obstacles, breakthroughs.

## Taste And Craft Lab

Used for cooking, styling, writing, design, photography, or making.

Common loop: attempt -> sensory judgment -> adjustment -> personal preference map.

Useful memory: preferences, constraints, experiments, successes, disliked patterns.

## Thinking Lab

Used for research, reading, strategy, philosophy, or complex decisions.

Common loop: question -> source or thought capture -> synthesis -> next inquiry.

Useful memory: open questions, claims, evidence, uncertainties, discarded paths.

## Care And Maintenance Lab

Used for health routines, home care, relationships, finances, or energy management.

Common loop: check-in -> action -> consequence -> adjustment.

Useful memory: boundaries, triggers, obligations, recurring maintenance needs.

## Identity Reflection Lab

Used for values, transitions, grief, ambition, confidence, or meaning-making.

Common loop: lived moment -> reflection -> theme -> experiment or commitment.

Useful memory: language that resonates, avoided questions, recurring tensions, chosen experiments.
